# Couple App - 情侣日常应用

## 项目介绍

一款专为情侣设计的日常互动 App，帮助情侣记录美好时光、保持实时连接、增添生活仪式感。

## 核心功能清单

### 1. 💬 聊天
- 私信功能
- 文字、图片、表情包

### 2. 📅 纪念日管理
- 相识日期、相恋日期记录
- 重要节日倒计时（情人节、周年等）
- 自定义纪念日添加

### 3. 📸 共享相册
- 双方照片/视频共享
- 按时间线浏览
- 相册分类

### 4. 💕 心情打卡
- 每日心情记录（开心/一般/难过等）
- 历史心情日历查看
- 心情趋势统计

### 5. 🗺️ 实时位置共享
- 地图上显示双方实时位置
- 位置共享开关（可随时关闭）
- 到达/离开地点提醒
- 位置共享历史记录

### 6. 📝 一起写日记
- 共同编辑日记本
- 按日期记录
- 支持图文混排

### 7. 🎁 愿望清单
- 礼物愿望清单
- 已实现/待实现分类
- 互相查看对方的愿望

### 8. 🔔 纪念日提醒
- 重要日子推送提醒
- 定时发送祝福消息
- "在一起 X 天"每日推送

## 技术选型

| 类别 | 技术方案 |
|------|----------|
| 跨平台框架 | Flutter |
| 地图/定位 | 高德地图 SDK |
| 后端服务 | LeanCloud / Firebase BaaS |
| 存储 | 端对端加密 + 云存储 |
| 推送 | 极光推送 / Firebase Cloud Messaging |

## 隐私说明

- 位置共享需双方同意，可随时关闭
- 聊天记录端对端加密
- 数据存储遵循隐私最小化原则
